# Changelog

All notable changes to azd-rest will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of azd-rest extension
- Support for GET, POST, PUT, PATCH, DELETE HTTP methods
- Automatic Azure authentication via azd
- Azure context integration (subscription, tenant, environment)
- Custom header support
- Request body from inline data or file
- Response formatting (JSON pretty-print)
- Output to stdout or file
- Verbose mode for debugging
- TLS verification with optional skip
- Comprehensive test suite
- CI/CD pipeline with linting, testing, spell checking, and security scanning
- Multi-platform builds (Windows, Linux, macOS)
- Full documentation (README, SPEC, CONTRIBUTING)

## [0.1.0] - 2025-11-08

### Added
- Initial implementation of azd-rest extension
- Core REST API functionality
- Azure Developer CLI integration
- Comprehensive documentation
- Automated testing and CI/CD
