azd rest get

Execute a GET request

Usage

rest get <url> [flags]

Flags

Flag Short Description
--help -h help for get
--binary - Stream request/response as binary without transformation
--data -d string Request body (JSON string)
--data-file - string Read request body from file (also accepts @{file} shorthand)
--follow-redirects - Follow HTTP redirects (default true)
--format -f string Output format: auto, json, raw (default "auto")
--header -H stringArray Custom headers (repeatable, format: Key:Value)
--insecure -k Skip TLS certificate verification
--max-redirects - int Maximum redirect hops (default 10)
--no-auth - Skip authentication (no bearer token)
--output-file - string Write response to file (raw for binary content)
--paginate - Follow continuation tokens/next links when supported
--retry - int Retry attempts with exponential backoff for transient errors (default 3)
--scope -s string OAuth scope for authentication (auto-detected if not provided)
--timeout -t duration Request timeout (default 30s)
--verbose -v Verbose output (show headers, timing)
